## 3. PAGE RECIPES

Winning section structures per page type, ordered top-to-bottom.

### 3A. Advertorial Page (Editorial News Style)

**Used by**: tryemsense, tryledisa, halogrow, vibriance, smoothspire, drivse, clarifion, oricle

```
1.  SALE BANNER / TOP BAR              [NAV-01] -- bg: navy or lime, text: white or dark
2.  LOGO HEADER                        [NAV-02] -- simple logo, optional "Advertorial" label
3.  (optional) BREADCRUMB              [NAV-03] -- "Health > Wellness > ..."
4.  (optional) AUTHOR BYLINE + DATE    [HERO-03] -- avatar 30-40px + name + separator + date
5.  HERO HEADLINE (H1)                 [HERO-01] -- 36-42px/800, attention-grabbing
6.  HERO IMAGE / VIDEO                 [HERO-02] -- featured media, 16:9
7.  BODY TEXT -- Problem Section       [BODY-01] -- pain point narrative
8.  INLINE CTA BUTTON                  [CTA-01] -- first CTA after problem
9.  BODY TEXT -- Solution Section      [BODY-01] -- product introduction
10. BENEFIT / FEATURE LIST             [BODY-03] -- bulleted benefits with checkmarks
11. IMAGE GALLERY / PROOF              [BODY-06] -- before/after or product shots (10+ images)
12. INLINE CTA BUTTON                  [CTA-01] -- second CTA mid-article
13. TESTIMONIALS / REVIEWS             [PROOF-01] -- customer quotes
14. NUMBERED LIST / REASONS            [BODY-02] -- "5 Reasons Why..." pattern
15. (optional) COMPARISON TABLE        [BODY-09] -- product vs competitors
16. (optional) BEFORE/AFTER            [BODY-10] -- visual transformation
17. SOCIAL PROOF / EXPERT QUOTES       [PROOF-02] -- doctor/expert endorsement
18. GUARANTEE / WARRANTY               [CTA-04] -- money-back badge
19. URGENCY / COUNTDOWN                [CTA-03] -- limited stock/timer
20. (optional) PRICING / BUNDLE CARDS  [CTA-02] -- at bottom or sidebar
21. (optional) FAQ SECTION             [FORM-06] -- accordion-style
22. FOOTER / DISCLAIMERS               [FOOT-01,02] -- legal, links, copyright
23. MOBILE STICKY BAR                  [NAV-05] -- fixed bottom CTA on mobile
```

**Layout**: 2-column editorial on desktop (content 760px + sidebar 300px), 1-column on mobile. Max-width 1100px.

### 3B. Product/Sale Page (Long-Form Sales)

**Used by**: airmoto, oricle, drivse, rejuvacare, smoothspire, tryemsense, tryledisa, getheyfra, primalqueen

```
1.  ANNOUNCEMENT BAR                   [NAV-01] -- "70% OFF + FREE SHIPPING"
2.  (optional) TIMER NAV BAR           [NAV-06] -- sticky top, countdown + anchor links
3.  HERO SECTION                       [HERO-05] -- product image + headline + rating + video
4.  USP BAR                            [PROOF-08] -- "100,000+ verified users" + 3-4 badges
5.  (optional) AS SEEN ON              [PROOF-07] -- media logos row
6.  PROBLEM / AGITATION                [BODY-01] -- pain point narrative, dark bg variant
7.  VIDEO SECTION                      [BODY-08] -- product demo or explainer
8.  BENEFITS / FEATURES                [BODY-03] -- 3-5 key benefits with icons
9.  (optional) INGREDIENTS             [BODY-12] -- individual ingredient/feature cards
10. HOW IT WORKS (3 steps)             [BODY-11] -- step cards with icons
11. RESULTS / STATS                    [PROOF-05] -- large number + bar charts
12. TESTIMONIALS                       [PROOF-01,03] -- customer review cards
13. (optional) COMPARISON              [BODY-09] -- product vs competitors grid
14. PRICING / BUNDLE SELECTOR          [CTA-02] -- 3-column (1x, 3x, 6x) with "Best Value"
15. GUARANTEE                          [CTA-05] -- 30/90 day badge + text
16. FAQ                                [FORM-06] -- 5-10 questions accordion
17. FOOTER                             [FOOT-03,04,05] -- dark bg, 2-column, payment badges
18. STICKY BOTTOM BAR (mobile)         [NAV-05] -- fixed bottom CTA
```

**Layout**: Full-width sections, alternating backgrounds (white > light gray > cream > white). Max-width 1110-1250px for content.

**Background alternation pattern**:
1. White `#FFFFFF`
2. Light gray `#F5F5F5` or Light blue `#F4F9FF`
3. White `#FFFFFF`
4. Cream `#F9F2E8` or Beige `#EDEDE4`
5. White `#FFFFFF`

**Product page specific alternation (Webflow/int- pattern)**:
1. Cyan `#EAFAFA` (hero)
2. White `#FFFFFF` (features)
3. Light gray `#F5F5F5` (product details)

## 6. AI COMPOSITION RULES

Rules for an AI to compose new pages from these blocks and tokens.

### 6A. Page Type Selection

1. **Determine page type first**: Advertorial, Product, Checkout, Upsell, VSL, Thank You, or Landing
2. Select the matching recipe from Section 3
3. Choose brand colors from the appropriate color set in Section 2A

### 6B. Block Composition Order

1. **Always start** with NAV-01 (announcement bar) unless it is a Thank You page
2. **Always end** with FOOT-01 + FOOT-02 (footer links + disclaimer)
3. **CTA frequency**: Place CTA-01 every 3-5 content blocks (minimum 3 CTAs per page)
4. **Proof frequency**: Place at least 1 social proof block before each CTA
5. **Alternating backgrounds**: Never place two same-bg sections adjacent

### 6C. Color Selection Rules

1. **Default to the "Green Brand" color set** unless brand identity demands otherwise
2. CTA color MUST be one of: Green (#00C249, #24AA2F), Teal (#12AB88), or Blue (#1B96D3)
3. Never use orange or yellow as primary CTA unless brand is health/supplement VSL
4. Urgency always uses: Yellow bg (#FEFBC3) + Red border (#D0021B dashed)
5. Footer always uses: Dark bg (#141619) + darker copyright bar (#0E0F11)

### 6D. Typography Rules

1. **Heading font**: Montserrat (checkout), Open Sans (editorial), Sentient (premium product)
2. **Body font**: Always Open Sans or Roboto, 18-20px, 400 weight, 26px line-height
3. **H1**: 36-42px desktop, 22-28px mobile, weight 800
4. **CTA text**: 20-24px, weight 700-800, white color
5. **Price text**: 55-84px for hero price, 24-30px for per-unit, always bold

### 6E. Spacing Rules

1. **Section padding**: 64-80px vertical desktop, 16-40px mobile
2. **Container max-width**: 1100px (2-col) or 760px (1-col editorial)
3. **Grid gap**: 12px between columns, 8px between pricing cards
4. **CTA button**: Full-width within container, max 410px for editorial inline CTAs
5. **Mobile**: All padding reduces to 16px sides, 40px vertical

### 6F. Mobile Responsive Rules

1. **Breakpoint**: 768px (collapse to single column)
2. **Typography**: All sizes reduce 30-40% on mobile
3. **Layout**: 2-column becomes 1-column stack
4. **Sidebar**: Hidden or moved below content
5. **Sticky bar**: Hidden on desktop, appears fixed bottom on mobile
6. **Video**: 400px desktop height, 250px mobile height
7. **Pricing grid**: 3-column becomes single-column scroll or 2-column compact

### 6G. Checkout-Specific Rules

1. **Bundle selector**: Always 2+ options, middle option highlighted as "Most Popular" or "Best Value"
2. **"Most Popular" badge**: Red (#EC0B43) rotated ribbon, or amber (#F59E0B) top bar
3. **Express checkout**: Always above credit card form, separated by "OR" line
4. **Form order**: Email -> Shipping -> Payment -> Privacy checkbox -> CTA
5. **CTA style**: Green (#00C249), full-width, bold Montserrat, "ORDER NOW" or "COMPLETE PURCHASE"
6. **Order bump**: Cream bg (#FCF8E3), dashed border, toggle switch or checkbox
7. **Trust badges**: Always below CTA -- SSL, payment icons, guarantee

### 6H. Upsell-Specific Rules

1. **Interrupt pattern**: "WAIT!" header + countdown timer (10 min)
2. **CTA text**: "YES! Upgrade My Order" (positive), "No thanks, I don't want..." (negative)
3. **Negative option**: Blue link (#0000EE), small font (12-14px), long guilt-trip copy
4. **Price format**: Strikethrough old price (gray #808080) + new price + "% OFF" badge
5. **Guarantee**: Always mention "100% Money Back Guarantee" on upsell CTA
6. **Maximum 5 OTOs** in sequence, last one is package protection ($4.95)

### 6I. Accessibility and Trust Requirements

1. **Form inputs**: Always have visible labels or descriptive placeholders
2. **Privacy text**: Below checkout CTA, 14px gray (#818997), terms + privacy links
3. **Footer disclaimer**: Always present, centered, 50% opacity text
4. **Payment icons**: Visa, Mastercard, Amex, PayPal (always visible near CTA)
5. **SSL/Secure badges**: Always present in checkout footer or below CTA

### 6J. Animation Usage Rules

1. **CTA pulse**: Use sparingly, only on final/bottom CTA to draw attention
2. **Marquee**: Use for logo rows and testimonial carousels only
3. **Fade in**: Use for FOMO notifications (social proof popups)
4. **Hover effects**: All buttons MUST have hover state (color darken + optional scale)
5. **Transitions**: 0.2s for buttons, 0.3s for accordions, never more than 0.6s
